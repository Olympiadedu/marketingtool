ⓘ 토큰 발급 가이드
① Instagram 계정 준비
  1. Instagram 앱 → 프로필 → 설정 → 계정 → 비즈니스 계정으로 전환
  2. Facebook 페이지 연결 (기존 페이지 or 새로 생성)
② Facebook 앱 & 토큰 발급
  1. Facebook 개발자 → 새 앱 생성 (유형: Business)
  2. 앱 대시보드 → 제품 추가 → Instagram Graph API
  3. 앱 설정 → 기본 설정 → 앱 ID / 앱 시크릿 복사
  4. 그래프 API 탐색기 → 토큰 생성
    권한 추가: instagram_basic, instagram_content_publish, pages_read_engagement
  5. 단기 토큰을 장기 토큰으로 교환 (아래 URL 형식):
    https://graph.facebook.com/v19.0/oauth/access_token?grant_type=fb_exchange_token&client_id={앱ID}&client_secret={앱시크릿}&fb_exchange_token={단기토큰}
  6. 사용자 ID 확인: GET /me?fields=id&access_token={토큰}
③ GitHub Personal Access Token
  1. GitHub 토큰 발급 페이지 열기
  2. repo 권한 체크 → Generate token
  3. 생성된 토큰(ghp_...) 복사 후 위에 입력
  4. 이미지는 게시 완료 후 자동 삭제됩니다
▲ 액세스 토큰은 60일 후 만료됩니다. 만료 전 재발급이 필요합니다.

ⓘ 설정 방법
  1. 카카오 디벨로퍼스 → 앱 설정 → 플랫폼 키
  2. REST API 키 카드에서 키 복사 → 위 입력란
